# Adventure UI (Vietnamese Translation)

Giao diện Tiếng Việt cho card SillyTavern "Mô Phỏng Bang Hội Thám Hiểm" (冒险者公会模拟器).

Translated from original: https://ariespo.github.io/adventure

## Setup

1. Push repo này lên GitHub
2. Vào **Settings → Pages → Source: Deploy from branch → main → / (root)**
3. Chờ vài phút, GitHub Pages sẽ host tại: `https://YOUR_USERNAME.github.io/adventure-vi/`
